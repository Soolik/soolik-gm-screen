# Soolik's GM Screen

A toolkit for Pathfinder 2e Game Masters on Foundry VTT. Everything you run from
behind the screen, in one module. Actively maintained, English and Polish.

Compatible with Foundry VTT v13 and v14 (tested on 14.363), Pathfinder 2e system.

> The believable-fake-item trick nobody else has, the only investigation board your
> players build together in real time, and a merchant generator that actually looks
> good.

## Free and Premium

**Free** (in the Foundry module browser):

- **Accessibility Controls** — per-player brightness, contrast and saturation on the
  map canvas, with presets. Each player sets their own.
- **Whisper Toast** — a top-center banner so you never miss a whisper, with a
  countdown bar and optional sound.
- **Whisper Autocomplete** — type `/w` and get a dropdown of player and target names
  to complete the whisper.
- **Control Panel** — one window listing every feature with on/off toggles, plus an
  in-app user guide.

**Premium** (a Ko-fi subscription, then a separate add-on you install):

- **Misidentify** — turn a critically failed Identify into a convincing fake the
  player will actually believe. The truth stays GM-only; every swap is reversible.
- **Lore Checks** — tiered knowledge rolls where a better roll reveals more, a fumble
  feeds a believable false lead, and the dice stay secret.
- **Connection Map** — a shared investigation board styled like a tavern notice-board,
  built by your whole table in real time.
- **Emporium** — generate stocked merchants and level-appropriate treasure hoards in
  seconds, preview and reroll, then drop them on the map.

Premium includes every update while your subscription is active.

## Install

**Free module.** In Foundry: Add-on Modules, Install Module, and either find
"Soolik's GM Screen" in the browser or paste this manifest URL:

```
https://dist.wezprzerzuc.pl/free/module.json
```

**Premium add-on.** Subscribe on Ko-fi, then you receive a personal manifest link.
In Foundry, Install Module and paste that link. The premium features then appear in
the Control Panel, ready to enable.

Subscribe: https://ko-fi.com/soolik

## User guide
