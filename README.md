# Soolik's GM Screen

A pile of tools for running Pathfinder 2e on Foundry VTT, the stuff you deal with
from behind the screen. I build it for my own games and keep it updated because I
actually use it. English and Polish.

Works on Foundry VTT v14 (tested on 14.363), Pathfinder 2e system.

> The believable fake-item trick nobody else has, the only investigation board your
> players build together in real time, and a shop generator that doesn't look like a
> form from 2021.

## Free and premium

**Free** (just grab it from Foundry's module browser):

- **Accessibility Controls** brightness, contrast and saturation for the map, per
  player. Everyone sets their own, nobody else is affected.
- **Whisper Toast** a banner up top so you stop missing whispers. Countdown bar,
  optional sound.
- **Whisper Autocomplete** type `/w` and pick names from a dropdown instead of
  fighting the parser.
- **Control Panel** one window with all the toggles, plus the user guide built in.

**Premium** (a Ko-fi sub, then a small add-on you install):

- **Misidentify** a player botches an Identify, you hand them a fake they'll
  completely buy. The truth stays GM-only and every swap is reversible.
- **Lore Checks** tiered knowledge: a better roll reveals more, a fumble feeds a
  believable lie, and the dice stay secret.
- **Connection Map** a shared corkboard styled like a tavern notice-board, that your
  whole table builds together, live.
- **Emporium** stocked merchants and level-appropriate treasure in seconds. Preview,
  reroll, drop it on the map.

Premium includes every update while your sub is active.

## Install

**Free module.** In Foundry: Add-on Modules, Install Module, then either find
"Soolik's GM Screen" in the browser or paste this manifest URL:

```
https://dist.wezprzerzuc.pl/free/module.json
```

**Premium add-on.** Subscribe on Ko-fi and you get a personal manifest link. In
Foundry, Install Module, paste that link, and the premium features show up in the
Control Panel ready to switch on.

Subscribe: https://ko-fi.com/soolik
